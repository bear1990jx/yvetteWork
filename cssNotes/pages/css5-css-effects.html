box-shadow 盒子阴影

box-shadow：x坐标值（正右负左） y坐标值（正下负上） 模糊程度 阴影扩散长度 颜色

  • inset: 默认阴影在边框外。使用 inset 后，阴影在边框内（即使是透明边框），背景之上内容之下。
  • offset-x offset-y: 阴影偏移量。水平偏移量，如果是负值则阴影位于元素左边。两者都是0，那么阴影位于元素后面。这时如果设置了blur-radius或spread-radius有模糊效果。
  • blur-radius：值越大，模糊面积越大，阴影就越大越淡。 不能为负值。默认为0，此时阴影边缘锐利。
  • spread-radius：取正值时，阴影扩大；取负值时，阴影收缩。默认为0，此时阴影与元素同样大。
  • color：如果没有指定，则由浏览器决定——通常是color的值，不过目前Safari取透明。

text-shadow 文本阴影

text-shadow：x坐标值（正右负左） y坐标值（正下负上） 模糊半径 颜色

作用

  • 立体感 text-shadow: 1px 1px 0 #aaa;
  • 印刷品质感(边缘油墨模糊感) text-shadow: 0 0 1px rgba(128,128,128,.2);

border-radius

  • 圆角矩形
  • 圆形：border-radius: 50%;
  • 半圆、扇形
  • 一些奇怪的角

注意：border-radius写50%，即使有边框，也是圆形；如果写宽度一半的px值，加上边框就不是圆，因为半径其实是边框值加宽度的一半。

border-radius: x / y，分别对应水平方向和垂直方向的半径。等同，border-top-left-radius: x y;，写单个角不需要/。

background

  • 纹理、图案
  • 渐变
  • 雪碧图动画
  • 背景图尺寸适应

background-position: center center;指定背景图位置

background-repeat: repeat-x水平重复，repeat-y垂直重复，no-repeat不重复。

background-size: 50% 50%;，百分比相对于容器大小。100% 100%就刚好覆盖，和容器一样大。
有可能图片会被拉伸，设置cover，覆盖整个容器画面，保持长宽比不变。
设置contain，图片会完整显示出来，多的地方空白。

clip-path

按路径进行裁剪

  • 对容器进行裁剪：指定容器一部分显示，一部分被裁掉。
  • 按常见几何图形裁剪
  • 自定义路径裁剪

和border-radius区别：容器占位大小不变，border-radius需调整容器的大小才能实现相同的效果。
clip-path不改变容器定位，可以实现容器内动画。
