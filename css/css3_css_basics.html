    选择器的作用:
  • 用于匹配HTML元素, 即页面中哪些元素要匹配这个样式
  • 有不同的匹配规则
  • 多个选择器可以叠加

选择器

  • 分类和权重(重要)
  • 解析方式和性能(可能出现在面试中, 开发中没有什么用)
    浏览器解析css方式是从右往左, 出于性能的考虑, 更快速的确定位置.
  • 值得关注的选择器

选择器的分类

  • 元素选择器 a{}
    指定元素的名字.
  • 伪元素选择器 ::before{}
    伪元素是before, after这样的元素, 不会出现在html中, 也不会出现在DOM树中, 但是是一个真实存在的元素, 可以显示内容, 设置样式.
  • 类选择器 .link{}
    如class
  • 属性选择器 [type=radio]{}
  • 伪类选择器 :hover{}
    伪元素是双冒号::, 伪类是单冒号:, 伪类是一个元素的状态.
  • ID选择器 #id{}
    id在页面中必须是唯一的, 一般用于选中一个特定的元素或区域, 不会用id选择一堆元素, 这样的html不合法.
  • 组合选择器 [type=checkbox] + label{}
  • 否定选择器 :not(.link){}
  • 通用选择器 *{} *表示所有元素都匹配

选择器的权重

  • ID选择器 #id{} +100
  • 类 属性 伪类 +10
  • 元素 伪元素 +1
  • 其他选择器 +0
